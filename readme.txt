fix-things version 4.1

This program fixes file extensions case,
converts between Shift_JIS and UTF-8 BOM,
and changes newline codes between \\r\\n and \\n.

このプログラムは対象のファイルに以下の修正を加えます。拡張子の表記統一、
文字コード、Shift_JISとUTF-8 BOMの統一、
改行コード「\\r\\n」と「\\n」の統一

Эта программа исправляет регистр расширений файлов,
преобразует кодировку файлов между Shift_JIS и UTF-8 BOM,
и преобразует код конца строки между \\r\\n и \\n.
